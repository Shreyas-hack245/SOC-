# 🛡️ SOC LAB
### Build • Attack • Detect • Investigate • Respond

A modern **Security Operations Center (SOC)** simulation platform designed to demonstrate enterprise cyber defense workflows through realistic attack simulations, live security monitoring, AI-assisted incident analysis, and interactive threat investigation.

---

## 📌 Overview

SOC LAB recreates the workflow of a professional Security Operations Center.


Instead of simply displaying logs or dashboards, the platform simulates the complete lifecycle of a cyber attack:


```
   Attacker
      │
      ▼
Attack Simulation
      │
      ▼
Detection Engine
      │
      ▼
Security Alerts
      │
      ▼
Threat Analysis
      │
      ▼
MITRE ATT&CK Mapping
      │
      ▼
AI Investigation
      │
      ▼
Incident Response
```

This project demonstrates how enterprise SOC teams detect, triage, investigate, and respond to security incidents using modern defensive technologies.

## ⚔ Attack Simulations

The dashboard simulates multiple offensive techniques including:

- Nmap Reconnaissance
- SYN Flood (DoS)
- Hydra SSH Brute Force
- Hydra FTP Brute Force
- SQL Injection
- Directory Traversal
- PowerShell Execution
- Encoded Payload Detection
- DNS Tunneling
- Suspicious RDP Activity

Each attack generates realistic SOC telemetry and investigation artifacts.
